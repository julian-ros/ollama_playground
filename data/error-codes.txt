Julio API Error Codes Reference

HTTP Status Codes and Error Messages

200 OK - Request successful
201 Created - Resource created successfully  
204 No Content - Request successful, no content returned
400 Bad Request - Invalid request parameters
401 Unauthorized - Invalid or missing API key
403 Forbidden - Insufficient permissions
404 Not Found - Resource not found
409 Conflict - Resource conflict (e.g., duplicate name)
413 Payload Too Large - Request body too large
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error - Server error
502 Bad Gateway - Upstream service error
503 Service Unavailable - Service temporarily unavailable

Common Error Response Format:
{
  "error": "TASK_001",
  "message": "The requested task does not exist",
  "code": 404,
  "details": {
    "task_id": "task_abc123",
    "timestamp": "2024-01-15T10:30:00Z"
  },
  "help_url": "https://docs.julio.com/errors/TASK_001"
}

Rate Limiting Headers:
X-RateLimit-Limit: 500
X-RateLimit-Remaining: 499
X-RateLimit-Reset: 1642248300

Retry Guidelines:
- 429 errors: Use exponential backoff
- 5xx errors: Retry after 1-5 seconds
- Network errors: Retry up to 3 times
- Task creation errors: Check for duplicates first
